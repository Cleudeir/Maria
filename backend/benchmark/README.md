# Maria Agent Benchmark Suite

This directory contains a suite of **30 test tasks** designed to evaluate the coding and file-creation accuracy of the **Maria agent** under controlled, predictable settings.

## Structure

- **`tasks.json`**: The database containing the 30 tasks, categorized by difficulty (10 simple, 10 medium, 10 expert).
- **`test_cases.py`**: Python assertion logic to verify the correctness of the code generated by the agent for each task.
- **`run_benchmark.py`**: Execution harness script that handles workspace isolation, runs the Maria agent, executes assertions, and compiles the final precision report.

## How to Run

Before running, ensure Ollama is running and has the model `qwen3.5:4b` downloaded (or modify target parameters).

### 1. Run all 30 Tasks
To evaluate the agent on all 30 tasks:
```bash
./benchmark/run_benchmark.py
```

### 2. Run a specific difficulty tier
To run only the simple tasks:
```bash
./benchmark/run_benchmark.py --difficulty simple
```

### 3. Run a single task by ID
To run task 15 (Password Generator):
```bash
./benchmark/run_benchmark.py --task 15
```

### 4. Run verification only
If you have already executed the tasks and want to re-run the verification tests without invoking the LLM:
```bash
./benchmark/run_benchmark.py --verify-only
```

## Report Analysis

The benchmark generates a markdown precision report at `benchmark/results/report.md`. This report details:
- **Total Success Rate**: The percentage of tasks that completed successfully without errors.
- **Verification Pass Rate (Precision)**: The percentage of tasks where the generated files compiled and passed all test assertions.
- **Time Elapsed**: Compilation of execution speed for each task.
- **Failures Detail**: Step errors and traceback details for debugging.
