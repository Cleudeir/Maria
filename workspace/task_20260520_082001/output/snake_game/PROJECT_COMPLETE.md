## 🎮 Game Controls

| Key | Action |
|-----|--------|
| UP | Move Up |
| DOWN | Move Down |
| LEFT | Move Left |
| RIGHT | Move Right |
| SPACE | Pause/Resume |
| ESC | Exit Game |

## 🏆 Game Rules

- Eat red food to grow and score points (+10)
- Avoid colliding with yourself
- Game ends on self-collision
- Press Space to pause/resume
