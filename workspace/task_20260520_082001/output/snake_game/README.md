# 🐍 Snake Game - Complete

A classic Snake game built with Python and Pygame using MVC architecture.

## 📋 Features

- ✅ Smooth 60 FPS gameplay
- ✅ Arrow key controls
- ✅ Score tracking (+10 per food)
- ✅ Self-collision detection
- ✅ Pause/Resume functionality
- ✅ Clean MVC architecture
- ✅ Pygame rendering
- ✅ Full test suite

## 🎯 Project Structure

```
output/snake_game/
├── main.py          # Entry point & game loop
├── game_config.py   # Configuration constants
├── models.py        # Snake, Food, Direction
├── controllers.py   # GameController logic
├── renderer.py      # Pygame rendering
├── tests/           # Test suite
├── README.md        # This file
└── PROJECT_COMPLETE.md # Project summary
```

## 🚀 Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Run the game
python main.py

# Run tests
python -m pytest tests/ -v
```

## 🎮 Controls

| Key | Action |
|-----|--------|
| UP | Move Up |
| DOWN | Move Down |
| LEFT | Move Left |
| RIGHT | Move Right |
| SPACE | Pause/Resume |
| ESC | Exit |

## 🏗️ Architecture

This project follows the MVC (Models-Views-Controllers) pattern:

- **Models**: Snake, Food, Direction classes
- **View**: Renderer (Pygame)
- **Controller**: GameController logic
